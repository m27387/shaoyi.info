Before Start
#

There are GUI applications from GitHub for both Windows and macOS. For CLI, users can use Homebrew to install Git easily on the macOS system. For the record, I switched from Sublime Text to VS Code, which works perfectlly well for me.

Hop on
#

Git is all about version control. Developer should set up the name and email for later recognition. Also set up the line coloring for easy reviewing. 

git config --global user.name "[firstname lastname]"
git config --global user.email "[valid-email]"
git config --global color.ui auto

Initialization
#

The first step at this phase is to establish a local repository for your project by git init. It is always better to stand on the shoulders of the giants by using git clone [url] to copy a sound project and modify it.
